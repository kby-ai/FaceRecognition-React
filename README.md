## 🧰 How to Run
- Install `npm` and `node` package and then check the version with the following commands.
```bash
npm --version
node --version
```
- Install dependencies
```bash
npm install
```
- Copy the pre-trained model weights
```bash
node post-install.js
```
- Run project
```bash
npm start
```
